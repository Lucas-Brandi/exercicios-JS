
## 📂 Resumo dos exercícios

### ⏱️ Contador
Um contador de números que pode aumentar,diminuir ou zerar, a depender do que for clicado pelo usuário

---
### ☀️🌑 Site com dark-mode
Um site que pode alternar entre modo claro e modo escuro

---
###🧮 Calculador de Imc
